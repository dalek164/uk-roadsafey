UK road law essentials

The Highway Code is the foundation for safe travel across the UK. It applies to everyone on the road.

  • Speed limits: obey local limits, especially in built-up areas, near schools and on motorways.
  • Priority: give way when required and know the rules at roundabouts and junctions.
  • Seat belts: everyone must wear a seat belt if one is fitted.

Highway Code updates are highlighted so the latest changes are easy to spot for 16–25 year olds.

Highway Code updates this year

These recent amendments help protect walkers, cyclists and anyone using new road layouts.

  • Stronger guidance on giving way to pedestrians at crossings and junctions.
  • Clearer rules for cyclists and powered scooters on shared routes.
  • Enhanced advice for young drivers about hazards, road signs and safe speed.

Safe space for vulnerable users

Watch for pedestrians, cyclists and motorcyclists. Keep a safe distance and pass only when it is safe.

Phones and distractions

Using a handheld phone while driving is illegal. Plan routes and stop safely before checking your phone.

Stay up to date with the Highway Code and follow local rules wherever you travel.
